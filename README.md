# NEON-GROUND

온라인 카지노 시뮬레이터. 실제 머니는 사용하지 않습니다.

🚧 현재 개발 중인 미완성 프로젝트입니다.

**[배포 링크](https://neon-ground.vercel.app)**

---

## 기능

- 블랙잭 — WebSocket 기반 실시간 게임
- 룰렛 — 멀티플레이어 실시간 베팅, 채팅
- 포커 — 멀티플레이어 텍사스 홀덤 🚧
- 회원가입 / 로그인 / 회원탈퇴 (Soft Delete)
- 프로필 — 게임 통계, 정보 수정
- 랭킹 — 수익/잔액/승률/게임 수 정렬
- 보너스 머니
